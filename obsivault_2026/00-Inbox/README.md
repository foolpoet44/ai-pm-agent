# 📥 Inbox - 원본 문서 수집함

## 용도
회장/CEO/임원 신년사, 지시사항, 전략 문서 등 원본 자료를 수집하는 폴더입니다.

## 파일 저장 규칙
- **파일명 형식**: `[출처]-[발신자]-[날짜].확장자`
- **예시**:
  - `신년사-회장-2025-01-01.pdf`
  - `CEO-Letter-2025-01-01.md`
  - `임원회의-지시사항-2025-01-10.docx`

## 지원 파일 형식
- PDF
- DOCX, DOC
- Markdown (MD)
- 텍스트 (TXT)
- 이미지 (JPG, PNG) - OCR 필요시

## 처리 프로세스
1. **수집**: 이 폴더에 원본 문서 저장
2. **분석**: Claude에게 문서 분석 요청
3. **추출**: Strategic Message로 구조화
4. **이동**: 분석 완료 후 `01-Strategic-Messages/`로 이동

## Claude 프롬프트 예시

```
[Inbox의 파일을 읽어서]

위 문서를 분석하여 다음을 수행해주세요:

1. Templates/strategic-message.md 템플릿 사용
2. 핵심 메시지 3-5개 추출
3. 전략 테마 분류
4. 명시적/암묵적 목표 도출
5. 타임라인 및 우선순위 파악

결과를 01-Strategic-Messages/[파일명].md로 저장해주세요.
```

## 보관 정책
- 분석 완료된 원본은 `00-Inbox/Archive/YYYY/` 폴더로 이동
- 최소 2년간 보관
- 중요 문서는 영구 보관
