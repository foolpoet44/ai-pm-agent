## 🚀 즉시 사용 가능

### 방법 1: Obsidian에서 바로 시작

1. **다운로드한 폴더 열기**
   ```
   Obsidian → Open folder as vault
   → Strategic-Planning-2025 선택
   ```

2. **첫 문서 작성**
   ```
   00-Inbox에 신년사 PDF 추가
   → Claude에게 분석 요청
   → 01-Strategic-Messages에 저장
   ```

3. **Dashboard 확인**
   ```
   06-Progress-Tracking/Dashboard-2025.md 열기
   → 실시간 현황 확인
   ```
