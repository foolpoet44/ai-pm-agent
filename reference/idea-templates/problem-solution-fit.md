# Problem-Solution Fit Framework

## Phase 1: Problem Identification

### 1.1 Problem Statement
**Problem Definition:**
- What is the core problem?
- Who experiences this problem?
- When does this problem occur?
- Why is this a problem worth solving?

### 1.2 Problem Validation
**Evidence:**
- Customer interviews (min. 10)
- Market research data
- Competitor analysis

### 1.3 Problem Prioritization
**Criteria:**
- Frequency: How often does it occur?
- Severity: How painful is it?
- Market Size: How many people have this problem?
- Willingness to Pay: Will they pay to solve it?

## Templates

### Customer Interview Script
```
1. Can you describe the last time you experienced [problem]?
2. How did you try to solve it?
3. What didn't work?
4. How much time/money did you spend?
5. If there was a solution that could [solution], would you use it?
6. How much would you pay for it?
```

### Solution Validation Survey
```
1. Does this solution address your problem? (Yes/No)
2. How likely are you to use this? (1-10)
3. What concerns do you have?
4. What features are most important?
5. What would you pay for this?
```
